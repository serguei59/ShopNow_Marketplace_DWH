# Plateforme Data Marketplace – Documentation Technique

Bienvenue dans la documentation complète de la plateforme Data Marketplace.  
Elle regroupe :

- l'analyse du contexte et du besoin,
- le diagnostic de l'existant,
- la modélisation des situations de travail,
- l'architecture cible (technique, sécurité, MCO),
- le plan de gouvernance des données,
- les pipelines d’ingestion,
- le DRP et les bonnes pratiques,
- les éléments de preuve pour l’évaluation E6.

Cette documentation est générée via **MkDocs + Material for MkDocs** et structurée pour répondre aux attendus de la certification (E5/E6).
